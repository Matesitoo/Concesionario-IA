# API Concesionario de Autos

API completa para gestión de concesionario construida con FastAPI, compatible con SQLite (local) y Supabase (producción).

## Tecnologias

- **FastAPI** - Framework web
- **SQLAlchemy** - ORM
- **SQLite** - Base de datos local
- **Supabase** - Base de datos en producción
- **Vercel** - Despliegue
